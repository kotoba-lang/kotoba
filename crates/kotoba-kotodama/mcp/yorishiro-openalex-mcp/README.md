# @etzhayyim/yorishiro-openalex-mcp

MCP server exposing the **openalex** yorishiro (kami: `api.openalex.org`) as tools.

Per **ADR-2605211900**. Generated by `@etzhayyim/yorishiro` v0.1.0.

## Tools

| Tool | Purpose |
|---|---|
| `search_works` | Search OpenAlex works (papers, preprints) |
| `search_authors` | Search OpenAlex authors |

Each tool's input schema is the zod 4 mirror of the matching lexicon at
`00-contracts/lexicons/ai/etzhayyim/yorishiro/openalex/<op>.json`.

## Stdio (desktop)

```json
{
  "mcpServers": {
    "etzhayyim-yorishiro-openalex": {
      "command": "node",
      "args": ["/path/to/repo/20-actors/kotoba-kotodama/mcp/yorishiro-openalex-mcp/src/cli.ts"],
      "env": {
        "YORISHIRO_OPENALEX_BASE_URL": "https://api.openalex.org"
      }
    }
  }
}
```

## Streamable HTTP

```ts
import { createYorishiroOpenalexMcpServer } from "@etzhayyim/yorishiro-openalex-mcp";
import { createDefaultOpenalexHandle } from "@etzhayyim/yorishiro-openalex-mcp/handle";
import { StreamableHTTPServerTransport } from "@modelcontextprotocol/sdk/server/streamableHttp.js";

const handle = createDefaultOpenalexHandle({ baseUrl: process.env.YORISHIRO_OPENALEX_BASE_URL });
const server = createYorishiroOpenalexMcpServer({ handle });
const transport = new StreamableHTTPServerTransport({ sessionIdGenerator: () => crypto.randomUUID() });
await server.connect(transport);
```
