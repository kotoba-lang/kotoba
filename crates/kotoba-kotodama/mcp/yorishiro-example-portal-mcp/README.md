# @etzhayyim/yorishiro-example-portal-mcp

MCP server exposing the **example-portal** yorishiro (browser-only; base URL: `https://example.com`) as tools.

Per ADR-2605211900. Generated by `@etzhayyim/yorishiro` in browser-only mode.

## Tools

| Tool | Purpose |
|---|---|
| `read_heading` | Read the H1 heading from example.com |
| `search_term` | Fill a hypothetical search input on the portal and read results |

## Runtime requirement

```bash
pnpm install
pnpm exec playwright install chromium
```

A Chromium browser is launched per request (no persistent context).
