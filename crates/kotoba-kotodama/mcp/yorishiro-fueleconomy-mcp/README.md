# @etzhayyim/yorishiro-fueleconomy-mcp

MCP server exposing the **fueleconomy** yorishiro (kami: `www.fueleconomy.gov`) as tools.

Per **ADR-2605211900**. Generated by `@etzhayyim/yorishiro` v0.1.0.

## Tools

| Tool | Purpose |
|---|---|
| `download_vehicles_csv` | Download the full EPA vehicles dataset as CSV |

Each tool's input schema is the zod 4 mirror of the matching lexicon at
`00-contracts/lexicons/ai/etzhayyim/yorishiro/fueleconomy/<op>.json`.

## Stdio (desktop)

```json
{
  "mcpServers": {
    "etzhayyim-yorishiro-fueleconomy": {
      "command": "node",
      "args": ["/path/to/repo/20-actors/kotoba-kotodama/mcp/yorishiro-fueleconomy-mcp/src/cli.ts"],
      "env": {
        "YORISHIRO_FUELECONOMY_BASE_URL": "https://www.fueleconomy.gov"
      }
    }
  }
}
```

## Streamable HTTP

```ts
import { createYorishiroFueleconomyMcpServer } from "@etzhayyim/yorishiro-fueleconomy-mcp";
import { createDefaultFueleconomyHandle } from "@etzhayyim/yorishiro-fueleconomy-mcp/handle";
import { StreamableHTTPServerTransport } from "@modelcontextprotocol/sdk/server/streamableHttp.js";

const handle = createDefaultFueleconomyHandle({ baseUrl: process.env.YORISHIRO_FUELECONOMY_BASE_URL });
const server = createYorishiroFueleconomyMcpServer({ handle });
const transport = new StreamableHTTPServerTransport({ sessionIdGenerator: () => crypto.randomUUID() });
await server.connect(transport);
```
