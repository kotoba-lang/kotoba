# @etzhayyim/yorishiro-clap-derive-demo-mcp

MCP server exposing the **clap-derive-demo** yorishiro (binary: `clap-derive-demo`).

Per ADR-2605211900. Generated by `@etzhayyim/yorishiro` in binary-cli mode.

## Tools

| Tool | Purpose |
|---|---|
| `encode` | clap-derive encode |
| `decode` | clap-derive decode |

## Runtime requirement

`clap-derive-demo` must be on the MCP host's PATH. The server
fails with `error: "binary not found on PATH"` if missing — no shell-out
attempts are made.

## Stdio (desktop)

```json
{
  "mcpServers": {
    "etzhayyim-yorishiro-clap-derive-demo": {
      "command": "node",
      "args": ["/path/to/repo/20-actors/kotoba-kotodama/mcp/yorishiro-clap-derive-demo-mcp/src/cli.ts"]
    }
  }
}
```
