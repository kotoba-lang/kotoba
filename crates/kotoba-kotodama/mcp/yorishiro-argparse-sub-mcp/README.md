# @etzhayyim/yorishiro-argparse-sub-mcp

MCP server exposing the **argparse-sub** yorishiro (binary: `argparse-sub`).

Per ADR-2605211900. Generated by `@etzhayyim/yorishiro` in binary-cli mode.

## Tools

| Tool | Purpose |
|---|---|
| `encode` | Encode an input file. |
| `decode` | Decode an input file. |
| `inspect` | Print metadata only. |

## Runtime requirement

`argparse-sub` must be on the MCP host's PATH. The server
fails with `error: "binary not found on PATH"` if missing — no shell-out
attempts are made.

## Stdio (desktop)

```json
{
  "mcpServers": {
    "etzhayyim-yorishiro-argparse-sub": {
      "command": "node",
      "args": ["/path/to/repo/20-actors/kotoba-kotodama/mcp/yorishiro-argparse-sub-mcp/src/cli.ts"]
    }
  }
}
```
