# @etzhayyim/yorishiro-argparse-multi-mcp

MCP server exposing the **argparse-multi** yorishiro (binary: `argparse-multi`).

Per ADR-2605211900. Generated by `@etzhayyim/yorishiro` in binary-cli mode.

## Tools

| Tool | Purpose |
|---|---|
| `main` | Standalone encoder driver. |
| `main_1` | Standalone decoder driver. |

## Runtime requirement

`argparse-multi` must be on the MCP host's PATH. The server
fails with `error: "binary not found on PATH"` if missing — no shell-out
attempts are made.

## Stdio (desktop)

```json
{
  "mcpServers": {
    "etzhayyim-yorishiro-argparse-multi": {
      "command": "node",
      "args": ["/path/to/repo/20-actors/kotoba-kotodama/mcp/yorishiro-argparse-multi-mcp/src/cli.ts"]
    }
  }
}
```
