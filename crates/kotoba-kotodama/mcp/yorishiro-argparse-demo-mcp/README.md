# @etzhayyim/yorishiro-argparse-demo-mcp

MCP server exposing the **argparse-demo** yorishiro (binary: `argparse-demo`).

Per ADR-2605211900. Generated by `@etzhayyim/yorishiro` in binary-cli mode.

## Tools

| Tool | Purpose |
|---|---|
| `main` | Demo argparse CLI used by the yorishiro source-repo fixture. |

## Runtime requirement

`argparse-demo` must be on the MCP host's PATH. The server
fails with `error: "binary not found on PATH"` if missing — no shell-out
attempts are made.

## Stdio (desktop)

```json
{
  "mcpServers": {
    "etzhayyim-yorishiro-argparse-demo": {
      "command": "node",
      "args": ["/path/to/repo/20-actors/kotoba-kotodama/mcp/yorishiro-argparse-demo-mcp/src/cli.ts"]
    }
  }
}
```
