# @etzhayyim/yorishiro-cobra-demo-mcp

MCP server exposing the **cobra-demo** yorishiro (binary: `cobra-demo`).

Per ADR-2605211900. Generated by `@etzhayyim/yorishiro` in binary-cli mode.

## Tools

| Tool | Purpose |
|---|---|
| `cobra_demo` | Demo cobra CLI used by the yorishiro source-repo fixture. |
| `greet` | Print a greeting. |
| `render` | Render output to file or stdout. |

## Runtime requirement

`cobra-demo` must be on the MCP host's PATH. The server
fails with `error: "binary not found on PATH"` if missing — no shell-out
attempts are made.

## Stdio (desktop)

```json
{
  "mcpServers": {
    "etzhayyim-yorishiro-cobra-demo": {
      "command": "node",
      "args": ["/path/to/repo/20-actors/kotoba-kotodama/mcp/yorishiro-cobra-demo-mcp/src/cli.ts"]
    }
  }
}
```
