# @etzhayyim/yorishiro-pdftotext-mcp

MCP server exposing the **pdftotext** yorishiro (binary: `pdftotext`).

Per ADR-2605211900. Generated by `@etzhayyim/yorishiro` in binary-cli mode.

## Tools

| Tool | Purpose |
|---|---|
| `convert` | Convert a PDF file to plain text |

## Runtime requirement

`pdftotext` must be on the MCP host's PATH. The server
fails with `error: "binary not found on PATH"` if missing — no shell-out
attempts are made.

## Stdio (desktop)

```json
{
  "mcpServers": {
    "etzhayyim-yorishiro-pdftotext": {
      "command": "node",
      "args": ["/path/to/repo/20-actors/kotoba-kotodama/mcp/yorishiro-pdftotext-mcp/src/cli.ts"]
    }
  }
}
```
