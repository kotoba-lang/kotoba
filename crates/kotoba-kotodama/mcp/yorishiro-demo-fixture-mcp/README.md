# @etzhayyim/yorishiro-demo-fixture-mcp

MCP server exposing the **demo-fixture** yorishiro (binary: `demo-fixture`).

Per ADR-2605211900. Generated by `@etzhayyim/yorishiro` in binary-cli mode.

## Tools

| Tool | Purpose |
|---|---|
| `greet` | Print a greeting for NAME. |
| `head` | Read up to MAX_LINES from INPUT_PATH and write to OUTPUT_PATH. |

## Runtime requirement

`demo-fixture` must be on the MCP host's PATH. The server
fails with `error: "binary not found on PATH"` if missing — no shell-out
attempts are made.

## Stdio (desktop)

```json
{
  "mcpServers": {
    "etzhayyim-yorishiro-demo-fixture": {
      "command": "node",
      "args": ["/path/to/repo/20-actors/kotoba-kotodama/mcp/yorishiro-demo-fixture-mcp/src/cli.ts"]
    }
  }
}
```
