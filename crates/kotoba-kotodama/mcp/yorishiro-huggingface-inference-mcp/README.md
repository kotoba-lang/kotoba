# @etzhayyim/yorishiro-huggingface-inference-mcp

MCP server exposing the **huggingface-inference** yorishiro (kami: `api-inference.huggingface.co`) as tools.

Per **ADR-2605211900**. Generated by `@etzhayyim/yorishiro` v0.1.0.

## Tools

| Tool | Purpose |
|---|---|
| `extract_features` | Run a feature-extraction (embedding) pipeline against a model |

Each tool's input schema is the zod 4 mirror of the matching lexicon at
`00-contracts/lexicons/ai/etzhayyim/yorishiro/huggingface-inference/<op>.json`.

## Stdio (desktop)

```json
{
  "mcpServers": {
    "etzhayyim-yorishiro-huggingface-inference": {
      "command": "node",
      "args": ["/path/to/repo/20-actors/kotoba-kotodama/mcp/yorishiro-huggingface-inference-mcp/src/cli.ts"],
      "env": {
        "YORISHIRO_HUGGINGFACE-INFERENCE_BASE_URL": "https://api-inference.huggingface.co"
      }
    }
  }
}
```

## Streamable HTTP

```ts
import { createYorishiroHuggingfaceInferenceMcpServer } from "@etzhayyim/yorishiro-huggingface-inference-mcp";
import { createDefaultHuggingfaceInferenceHandle } from "@etzhayyim/yorishiro-huggingface-inference-mcp/handle";
import { StreamableHTTPServerTransport } from "@modelcontextprotocol/sdk/server/streamableHttp.js";

const handle = createDefaultHuggingfaceInferenceHandle({ baseUrl: process.env.YORISHIRO_HUGGINGFACE-INFERENCE_BASE_URL });
const server = createYorishiroHuggingfaceInferenceMcpServer({ handle });
const transport = new StreamableHTTPServerTransport({ sessionIdGenerator: () => crypto.randomUUID() });
await server.connect(transport);
```
