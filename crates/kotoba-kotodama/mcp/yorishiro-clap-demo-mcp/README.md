# @etzhayyim/yorishiro-clap-demo-mcp

MCP server exposing the **clap-demo** yorishiro (binary: `clap-demo`).

Per ADR-2605211900. Generated by `@etzhayyim/yorishiro` in binary-cli mode.

## Tools

| Tool | Purpose |
|---|---|
| `clap_demo` | Demo clap CLI used by the yorishiro source-repo fixture. |

## Runtime requirement

`clap-demo` must be on the MCP host's PATH. The server
fails with `error: "binary not found on PATH"` if missing — no shell-out
attempts are made.

## Stdio (desktop)

```json
{
  "mcpServers": {
    "etzhayyim-yorishiro-clap-demo": {
      "command": "node",
      "args": ["/path/to/repo/20-actors/kotoba-kotodama/mcp/yorishiro-clap-demo-mcp/src/cli.ts"]
    }
  }
}
```
