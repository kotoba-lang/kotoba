# @etzhayyim/yorishiro-huggingface-mcp

MCP server exposing the **huggingface** yorishiro (kami: `huggingface.co`) as tools.

Per **ADR-2605211900**. Generated by `@etzhayyim/yorishiro` v0.1.0.

## Tools

| Tool | Purpose |
|---|---|
| `search_models` | Search HuggingFace Hub models |
| `search_datasets` | Search HuggingFace Hub datasets |

Each tool's input schema is the zod 4 mirror of the matching lexicon at
`00-contracts/lexicons/ai/etzhayyim/yorishiro/huggingface/<op>.json`.

## Stdio (desktop)

```json
{
  "mcpServers": {
    "etzhayyim-yorishiro-huggingface": {
      "command": "node",
      "args": ["/path/to/repo/20-actors/kotoba-kotodama/mcp/yorishiro-huggingface-mcp/src/cli.ts"],
      "env": {
        "YORISHIRO_HUGGINGFACE_BASE_URL": "https://huggingface.co"
      }
    }
  }
}
```

## Streamable HTTP

```ts
import { createYorishiroHuggingfaceMcpServer } from "@etzhayyim/yorishiro-huggingface-mcp";
import { createDefaultHuggingfaceHandle } from "@etzhayyim/yorishiro-huggingface-mcp/handle";
import { StreamableHTTPServerTransport } from "@modelcontextprotocol/sdk/server/streamableHttp.js";

const handle = createDefaultHuggingfaceHandle({ baseUrl: process.env.YORISHIRO_HUGGINGFACE_BASE_URL });
const server = createYorishiroHuggingfaceMcpServer({ handle });
const transport = new StreamableHTTPServerTransport({ sessionIdGenerator: () => crypto.randomUUID() });
await server.connect(transport);
```
