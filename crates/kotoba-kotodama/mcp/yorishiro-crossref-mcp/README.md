# @etzhayyim/yorishiro-crossref-mcp

MCP server exposing the **crossref** yorishiro (kami: `api.crossref.org`) as tools.

Per **ADR-2605211900**. Generated by `@etzhayyim/yorishiro` v0.1.0.

## Tools

| Tool | Purpose |
|---|---|
| `search_works` | Search Crossref works |
| `get_work_by_doi` | Get a Crossref work by DOI |

Each tool's input schema is the zod 4 mirror of the matching lexicon at
`00-contracts/lexicons/ai/etzhayyim/yorishiro/crossref/<op>.json`.

## Stdio (desktop)

```json
{
  "mcpServers": {
    "etzhayyim-yorishiro-crossref": {
      "command": "node",
      "args": ["/path/to/repo/20-actors/kotoba-kotodama/mcp/yorishiro-crossref-mcp/src/cli.ts"],
      "env": {
        "YORISHIRO_CROSSREF_BASE_URL": "https://api.crossref.org"
      }
    }
  }
}
```

## Streamable HTTP

```ts
import { createYorishiroCrossrefMcpServer } from "@etzhayyim/yorishiro-crossref-mcp";
import { createDefaultCrossrefHandle } from "@etzhayyim/yorishiro-crossref-mcp/handle";
import { StreamableHTTPServerTransport } from "@modelcontextprotocol/sdk/server/streamableHttp.js";

const handle = createDefaultCrossrefHandle({ baseUrl: process.env.YORISHIRO_CROSSREF_BASE_URL });
const server = createYorishiroCrossrefMcpServer({ handle });
const transport = new StreamableHTTPServerTransport({ sessionIdGenerator: () => crypto.randomUUID() });
await server.connect(transport);
```
