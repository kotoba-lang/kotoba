# @etzhayyim/yorishiro-bls-mcp

MCP server exposing the **bls** yorishiro (kami: `api.bls.gov`) as tools.

Per **ADR-2605211900**. Generated by `@etzhayyim/yorishiro` v0.1.0.

## Tools

| Tool | Purpose |
|---|---|
| `fetch_timeseries` | Fetch BLS timeseries data |

Each tool's input schema is the zod 4 mirror of the matching lexicon at
`00-contracts/lexicons/ai/etzhayyim/yorishiro/bls/<op>.json`.

## Stdio (desktop)

```json
{
  "mcpServers": {
    "etzhayyim-yorishiro-bls": {
      "command": "node",
      "args": ["/path/to/repo/20-actors/kotoba-kotodama/mcp/yorishiro-bls-mcp/src/cli.ts"],
      "env": {
        "YORISHIRO_BLS_BASE_URL": "https://api.bls.gov"
      }
    }
  }
}
```

## Streamable HTTP

```ts
import { createYorishiroBlsMcpServer } from "@etzhayyim/yorishiro-bls-mcp";
import { createDefaultBlsHandle } from "@etzhayyim/yorishiro-bls-mcp/handle";
import { StreamableHTTPServerTransport } from "@modelcontextprotocol/sdk/server/streamableHttp.js";

const handle = createDefaultBlsHandle({ baseUrl: process.env.YORISHIRO_BLS_BASE_URL });
const server = createYorishiroBlsMcpServer({ handle });
const transport = new StreamableHTTPServerTransport({ sessionIdGenerator: () => crypto.randomUUID() });
await server.connect(transport);
```
